## 安装

github [项目地址](https://github.com/MX8CLUB/TomatoHut)

本项目适用于相关技术人员学习交流，请自行编译安装

```sh
git clone https://github.com/MX8CLUB/TomatoHut.git

cd TomatoHut

yarn

react-native run-android
```
